Test Pilot 0.4

Test Pilot 0.4 is a maintenance release that fixes some minor problems, but more importantly it adds support for built-in surveys and for automatically recurring experiments. The next round of tests that we have planned will require both of these features.

A Week in the Life of a Browser »

This is a new study that will begin in December 2009, and recur periodically. It will run for a week each time, and collect data about the basic performance of Firefox — how often Firefox crashes or restarts, how many bookmarks it has, and so on. Find out more about this study here.
